毎月の流れ

毎月やることは大きく3つです。給料が入ったら・月の途中・月末に分けて覚えてください。

  • 給料が入ったら（25日以降）
    給料を記帳する

    代表口座(デビット)に収入として記帳します。

  • 給料記帳の後
    定例項目を計上する

    家賃・返済・積立など毎月決まった支出をまとめて記帳します。

  • 月中（日々）
    支出をそのつど記帳する

    コンビニや交通費など、使ったらすぐ記帳するのがおすすめです。

  • 月中（毎週）
    週次チェックをする

    レポートタブでその週の使い方をAIに評価してもらえます。

  • 月末
    月次確定をする

    その月の収支を締めて、自由貯金への繰越金額を確認します。

  • 月末（確定後）
    バックアップをとる

    データを保存しておきます。月に一度の習慣にしましょう。

🏠

ホーム

アプリを開くと最初に表示される画面です。おこづかいの残高と、各口座の状況が一目でわかります。

おこづかい残高

代表口座・PayPay・Suica・財布の合計残高です。今月使える残りのお金を示しています。

総資産

積立口座を含む全口座の合計残高です。

口座カード

各口座の残高と今月の使用額が表示されています。カードをタップすると記帳画面が開きます。

📈 増減を確認

各口座の残高が過去12ヶ月でどう変化したかをグラフで確認できます。口座を左右に切り替えて全口座を確認できます。

📅 定例を計上

毎月の定例項目（家賃・返済・積立）をまとめて記帳します。給料が入ったら（25日以降）実行してください（詳しくは「定例計上」の章）。

✏️

記帳のしかた

支出・収入・口座間の移動を記帳します。ホームで口座カードをタップすると記帳画面が開きます。

支出を記帳する

例：コンビニで買い物したとき

  • ホームで使った口座のカードをタップ
  • 「支出」タブが選ばれていることを確認
  • 金額・メモ・カテゴリ・日付を入力
  • 「記帳」ボタンを押す
カテゴリの選びかた
おこづかいの支出は、ガソリン代・タバコ代・定期代・医療費のいずれかを選びます。それ以外の日常的な支出は、食費・日用品・趣味・娯楽・交際費・雑費から選んでください。司法書士返済・借金返済(親へ)・家賃(親へ)は定例項目用のカテゴリです。

収入を記帳する

例：給料が入ったとき

  • ホームで代表口座(デビット)のカードをタップ
  • 画面上部の「収入」タブを選ぶ
  • 金額・メモ（例：「6月給与」）・日付を入力
  • 「記帳」ボタンを押す

口座間の移動を記帳する

例：代表口座からPayPayにチャージしたとき

  • ホームでいずれかの口座カードをタップ
  • 画面上部の「振替」タブを選ぶ
  • 移動元・移動先・金額・日付を入力
  • 「振替」ボタンを押す
⚠ 定例の積立は「振替」ではなく「定例計上」から行います
バイク保険やバイク点検などの月次積立は、給料が入ったら（25日以降）「定例を計上」ボタンからまとめて行ってください。ここの振替は、臨時の口座移動のためのものです。

記帳を間違えたとき

履歴タブで該当の記帳をタップすると、編集・削除ができます。
振替の場合は削除のみ（削除して入れ直してください）。

🔄

定例計上

給料が入ったら（25日以降）やること

家賃・返済・積立など毎月決まった支出を、まとめて記帳する機能です。給料が入ったら、最初にこれを行ってください。

ホーム › 📅 定例を計上
  • ホームの「📅 定例を計上」ボタンをタップ
  • 一覧が表示される。「全選択」ボタンで全件にチェックを入れる
  • 「選択した項目を計上」ボタンを押す
  • 計上日（実際に引き落とされた・振り込んだ日）を入力して「計上する」を押す
  • 全件が計上済みになると画面が自動的に閉じる
定例11件の内訳
司法書士返済（¥35,000）／家賃（¥50,000）／借金返済（¥10,000）
司法書士プール（¥15,000）／通信費（¥2,500）／バイク保険（¥3,500）
バイク点検（¥10,000）／バイク整備（¥3,000）／被服費（¥5,000）
生活防衛資金（¥10,000）／ライフイベント（¥5,000）
✓ 計上済みかどうかの確認
レポートタブ（月次確定）を開くと、定例計上パネルに各項目の状態が表示されています。
📊

集計タブ

月ごと・週ごとの支出をカテゴリ別に確認できます。「どこに使いすぎているか」が一目でわかります。

月次・週次の切り替え

上部の「月」「週」ボタンで切り替えます。月の場合は左右ボタンで対象月を変更できます。

カテゴリ別リスト

固定費・自由費・定例支払いなどの区分ごとに支出が表示されます。各カテゴリ行をタップするとドリルダウン（月別グラフ＋日別明細）が開きます。

CSV出力

リスト末尾の「CSV出力」ボタンから、その月の支出をスプレッドシートに書き出せます。

📋

レポートタブ・月次確定

週次チェックと月次確定の2つの機能があります。上部のタブで切り替えてください。

週次チェック

その週の支出をAIが評価してくれます。毎週使うことをおすすめします。

  • レポートタブ →「週次チェック」を選ぶ
  • 週セレクタで対象の週を確認（デフォルトは今週）
  • 「AIチェック」ボタンを押す → 少し待つとコメントが表示される

月次確定

月末に一度行います。その月の収支を締めて、自由貯金への繰越額を確認します。

  • レポートタブ →「月次確定」を選ぶ
  • 定例計上パネルで全件が計上済みか確認する
  • 詳細集計（固定費・自由費・定例支払いの内訳）を確認する
  • 「月次確定」ボタンを押す
  • 確認画面で繰越額・補充額を確認して「確定する」を押す
  • AI評価が生成される（少し待つ）
確定後にやること
確定が完了すると「自由貯金へ ¥〇〇 を移してください」と表示されます。 その金額を実際にSBI(自由貯金)に移動して、振替として記帳してください。

翌月のおこづかいが不足する場合は、不足分を代表口座から補充する振替も記帳します。
✓ 確定のやり直しもできます
記帳漏れに気づいた場合は、追記してから「確定し直す」ボタンで再確定できます。
📝

履歴タブ

すべての記帳が日付順に並んでいます。絞り込みや修正もここから行います。

絞り込み（フィルター）

上部の「口座▾」「日付▾」「カテゴリ▾」をタップして絞り込めます。青くなっているフィルターが有効です。「リセット」で全解除できます。

記帳の修正・削除

一覧の行をタップすると詳細が開き、編集・削除ができます。振替は削除のみ（修正は削除して入れ直し）。

CSV出力

右上の「CSV」ボタンから、年・月を指定して書き出せます。「月：指定なし」にすると1年分まとめて出力できます。

💬

相談タブ

購入を迷ったときのAI相談と、ほしいものリストの管理ができます。

購入相談

品名と価格を入力して「相談する」を押すと、今の残高をもとにAIが購入の可否やタイミングをアドバイスしてくれます。

ほしいものリスト

「ほしいもの」タブで品名と目標価格を登録しておくと、購入できるタイミングを確認しやすくなります。購入したら「購入した」ボタンでSBI(自由貯金)からの支出として記帳されます。

💾

バックアップ

月に一度

スマホのデータが消えてもバックアップがあれば復元できます。月次確定のあとに毎月行ってください。

設定 › バックアップ › JSONをダウンロード
  • 「設定」タブ →「バックアップ」を開く
  • 「JSONをダウンロード」を押す
  • ダウンロードされたファイルをiCloud DriveやGoogle Driveに保存する
⚠ Safariのデータを消去するとアプリのデータも消えます
「履歴とWebサイトデータを消去」を実行すると、アプリの記帳データも一緒に消えてしまいます。消去する前に必ずバックアップをとってください。

バックアップから復元する

設定 › バックアップ › JSONから復元
  • 「設定」タブ →「バックアップ」を開く
  • 「JSONから復元」を押す
  • 保存していたバックアップファイルを選択する
  • 確認ダイアログで「OK」を押す
⚠ 復元すると現在のデータはすべて上書きされます
復元前に現在のデータをバックアップとして保存しておくことをおすすめします。
